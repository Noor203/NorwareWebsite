Library system for East-Centre Library
"Norware were wonderful to work with, very good implementation of our requirements." - librarian at East-Centre Library

We were comissioned by a library to build a new system to replaced their pre-established paper-based system.

create prototypes (this project was heavily prototype-structured) and extensive testing by the staff.
This system was created using c++.
